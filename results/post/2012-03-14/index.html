Эхо вчерашнего взрыва

Про патенты и суд Яху с Фейсбуком. Постараюсь завтра про это уже не писать. Хотя нет, постараюсь держать вас в курсе :)

  • Некоторые пишут что Яху затеяло игру, в которой только один способ выиграть - не играть. В общем картина мира после апокалипсиса.
  • Яху пересек границу (у реки) - "но вообще, я их всех ненавижу, они ничего нового не придумали и вообще мертвы для меня", пишет Фред Уилсон из Union Square Ventures
  • Как Яху превратила мою работу в оружие - откровения Энди Бэйо, автора проекта upcoming.org, который был куплен Yahoo в 2005ом.
  • А Марк Кьюбэн согласен со мной. "Я надеюсь что Яху раздавит Фейсбук", пишет он. Но вообще пост полон сарказма :)

Браузеры и мобильные

  • Майкрософт намекнули, что Windows 8 будет запускать приложения для WP7. Кстати, похоже что слухи о полной несовместимости API WP7 и WP8 это правда.
  • Майкрософт предупреждает: после такого дождя ждите хороший отёл. Ну, в смысле, т.к. куча людей еще не накатили критический апдейт MS12-020, явно сейчас пройдет эпидемия нового червя. Дырка, если что, в RDP.
  • Сраница из 500+ сообщений, где европейские разработчики под андроид плачут. Нет, с платформой всё так, просто почему-то им не приходят деньги из Андроид Маркета.
  • Гугл может потерять дефолтный поиск в iPhone, не без оснований считает tnw.

Технологии

  • Боже! Наконец-то, нормальный синтаксис для инициализации некоторых ObjectiveC классов. Оцените пример:

NSArray *doctors = @[@"Matt Smith", @"David Tennant", @"Tom Baker"];

вместо старого

NSArray *doctors = [NSArray arrayWithObjects:@"Matt Smith", @"David Tennant", @"Tom Baker", nil];

кстати, по тексту примера тоже понятно, что автор - наш человек, да? Патчи в llvm, уже приняты в основную ветку.

  • Обнаружил для себя ql.io. Это такой метод для построения API и обращения к ним в стиле SQL. Буду изучать.
  • Гугл объявил сроки Google I/O - 27-29 июня, Сан-Франциско. Кстати, регистрация на Google Code Jam 2012 открыта.
  • Вышел второй релиз фреймворка Play!, единственный приемлимый для меня вариант, чтобы писать веб-приложения на Scala. Ну, благодаря принципам скалы, можно и на обычной Java использовать Play! Чуть подробнее в блоге TypeSafe
  • Метод решения проблем обращением к резиновой уточке. Хорошая статья, объясняющая как именно работает классический Rubber Duck Debugging.
  • Гит и Меркуриал. Чем хорош Git? пишут ребята из Atlassian. В общем, ребята, хана bitbucket'у!
